## Descrição

O Projeto Acelera é um projeto interdisciplinar desenvolvido com a orientação do professor Eduardo Enari e da professora Ana Carolina Satim, responsáveis pelas disciplinas de Estrutura de Dados e Programação Orientada a Objetos (POO), respectivamente.

O projeto foi elaborado em parceria com a Academia de Letras, Artes e História de Cruzeiro (ALAHC).

A aplicação foi desenvolvida exclusivamente na linguagem C# e, atualmente, não utiliza banco de dados.

O sistema tem como objetivo permitir o cadastro e visualização de obras e artistas, promovendo a cultura local da cidade de Cruzeiro.
